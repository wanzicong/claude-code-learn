### [0:23] 萤火虫（英维达）API 申请

**注册流程：**
1. 访问萤火虫官网，点击"获取一个API-KEY"
2. 输入邮箱、密码，创建账户
3. 完成邮箱验证、人机验证
4. 手机验证（中国 +86，发送短信验证）

**免费额度：**
- 不限量 API 请求
- 每分钟限制 44 个请求
- 支持 minimax-m2 和 glm4.7 模型

### [1:09] API Key 创建与测试

创建 API Key 后显示：
- 每分钟 44 个请求限制

### [1:15] Chatbox 配置测试

1. 打开 iStyle 6.6
2. 设置 → 模型服务 → 添加
3. 选择 OpenAI 方式
4. 输入 API Key 和密钥
5. 添加模型：minimax-m2、glm4.7

**测试结果：**
- minimax-m2：回复很慢，有时网络错误
- glm4.7：秒回，速度稳定
- 免费稳定性：时快时慢，取决于同时使用人数

### [2:20] 云服务器准备

**推荐配置：**
- 使用莱卡云（性价比高）
- 双核系统
- 建议开放 1-6535 端口（全部 TCP）
- 配置 SSH 密钥方便连接

**网络延迟：**
- 西安二线：30-35ms 延迟
- 其他厂商：40-45ms 延迟

## 关键要点

1. **免费 API**：萤火虫平台提供不限量 API 请求，但每分钟限制 44 次
2. **模型速度**：glm4.7 秒回稳定，minimax-m2 较慢且不稳定
3. **云服务器**：莱卡云性价比高，西安二线延迟 30-35ms
4. **部署方式**：推荐源码运行 + Claude 辅助，方便调试
5. **端口配置**：建议开放 1-6535 端口
6. **Docker**：可使用 Docker 容器化部署
7. **客户端**：国内用户推荐使用 Fay 开源项目

## 总结与建议

Moltbot 是一个功能强大的 AI 自动化机器人，通过萤火虫免费 API 可以降低使用成本。部署难度不大，但免费 API 稳定性较差，建议：

1. **测试阶段**：使用免费 API（萤火虫）
2. **生产环境**：考虑使用付费中转站 API 提高稳定性
3. **国内用户**：使用 Fay 开源项目作为客户端替代方案
4. **服务器**：选择国内云服务器（如莱卡云）降低延迟
