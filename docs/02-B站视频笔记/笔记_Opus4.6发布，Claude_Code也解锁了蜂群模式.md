### [02:29] 启用方式与显示模式

**[02:29-02:50] 配置方法**
在`.clinerules`文件中设置即可

**[02:34-03:07] 两种显示模式**

**1. In-process模式**
- 所有Teammate都在主终端里运行

**2. 窗格模式（Pane mode）**

### [02:50] Agent Teams的核心能力

**[02:50-03:24] 三大核心能力**
