# pyright-lsp

Python 语言服务器 (Pyright)，用于 Claude Code，提供静态类型检查和代码智能功能。

## 支持的扩展名
`.py`, `.pyi`

## 安装

通过 npm 全局安装 Pyright：

```bash
npm install -g pyright
```

或使用 pip：

```bash
pip install pyright
```

或使用 pipx（推荐用于 CLI 工具）：

```bash
pipx install pyright
```
