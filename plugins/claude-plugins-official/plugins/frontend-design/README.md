# 前端设计插件

生成独特、生产级的前端界面，避免通用的 AI 美学风格。

## 功能介绍

Claude 自动使用此技能处理前端工作。创建生产就绪的代码，具有：

- 大胆的美学选择
- 独特的排版和调色板
- 高影响力的动画和视觉细节
- 上下文感知的实现

## 使用方式

```
"Create a dashboard for a music streaming app"
"Build a landing page for an AI security startup"
"Design a settings panel with dark mode"
```

Claude 将选择清晰的美学方向，并以对细节的严谨关注实现生产代码。
