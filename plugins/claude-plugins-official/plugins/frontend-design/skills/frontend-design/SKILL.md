此技能指导创建独特、生产级的前端界面，避免通用的"AI 泛滥"美学。实现真正可用的代码，对美学细节和创意选择给予卓越的关注。

用户提供前端需求：要构建的组件、页面、应用程序或界面。他们可能包含关于目的、受众或技术约束的上下文。

## 设计思维

在编码之前，理解上下文并承诺一个大胆的美学方向：
- **目的**：这个界面解决什么问题？谁使用它？
- **基调**：选择一个极端：残酷极简主义、极度混乱主义、复古未来主义、有机/自然、奢华/精致、俏皮/玩具般、编辑/杂志风格、野兽派/原始风格、装饰艺术/几何、柔和/粉彩、工业/实用主义等。有太多风格可供选择。以这些为灵感，但设计一个符合美学方向的风格。
- **约束**：技术要求（框架、性能、可访问性）。
- **差异化**：什么让它令人难忘？人们会记住的一件事是什么？

**关键**：选择清晰的概念方向并精确执行。大胆的极致主义和精致的极简主义都有效——关键是有意图性，而不是强度。

然后实现可用的代码（HTML/CSS/JS、React、Vue 等），它需要：
- 生产级且功能完整
- 视觉震撼且令人难忘
- 具有清晰美学观点的连贯性
- 每个细节都经过精心打磨

## 前端美学指南

重点关注：
- **排版**：选择美观、独特、有趣的字体。避免 Arial 和 Inter 等通用字体；相反，选择能够提升前端美学的独特选择；出人意料、富有性格的字体选择。将独特的展示字体与精致的正文字体配对。
- **颜色与主题**：坚持连贯的美学。使用 CSS 变量保持一致性。带有鲜明强调色的主导色比胆怯、均匀分布的调色板效果更好。
- **动态**：使用动画实现效果和微交互。在 HTML 中优先使用纯 CSS 解决方案。在 React 中可用时使用 Motion 库。专注于高影响力的时刻：一个精心编排的页面加载，带有交错揭示（animation-delay）比分散的微交互产生更多愉悦。使用滚动触发和令人惊讶的悬停状态。
- **空间构成**：出乎意料的布局。不对称。重叠。对角线流动。打破网格的元素。慷慨的负空间或受控的密度。
- **背景与视觉细节**：创建氛围和深度，而不是默认为纯色。添加与整体美学匹配的上下文效果和纹理。应用创意形式，如渐变网格、噪声纹理、几何图案、分层透明度、戏剧性阴影、装饰边框、自定义光标和颗粒叠加。

永远不要使用通用的 AI 生成美学，如过度使用的字体家族（Inter、Roboto、Arial、系统字体）、陈词滥调的配色方案（特别是白色背景上的紫色渐变）、可预测的布局和组件模式，以及缺乏上下文特定个性的千篇一律设计。

创造性地解读并做出出乎意料的选择，感觉真正为上下文设计。每个设计都应该是不同的。在亮色和暗色主题、不同字体、不同美学之间变化。永远不要在代际之间收敛于常见选择（例如 Space Grotesk）。

**重要**：将实现复杂度与美学愿景匹配。极致主义设计需要精心制作的代码，包含大量动画和效果。极简主义或精致的设计需要克制、精确，并仔细关注间距、排版和微妙的细节。优雅来自于很好地执行愿景。
