# php-lsp

PHP 语言服务器 (Intelephense)，用于 Claude Code，提供代码智能和诊断功能。

## 支持的扩展名
`.php`

## 安装

通过 npm 全局安装 Intelephense：

```bash
npm install -g intelephense
```

或使用 yarn：

```bash
yarn global add intelephense
```
