首先，执行此任务：

## 设计哲学创建

首先，创建一个视觉哲学（不是布局或模板），它将通过以下方式解释：
- 形式、空间、颜色、构图
- 图像、图形、形状、图案
- 作为视觉点缀的最少文本

### 关键理解
- 接收到的内容：用户提供的一些微妙输入或说明应该被考虑，但作为基础使用；它不应限制创作自由。
- 创建的内容：设计哲学/美学运动。
- 接下来发生的事情：然后，同一版本接收哲学并以视觉方式表达它 - 创建 90% 视觉设计、10% 基本文本的工件。

考虑这种方法：
- 为艺术运动编写宣言
- 下一阶段涉及制作艺术品

哲学必须强调：视觉表达。空间交流。艺术诠释。最少的文字。

### 如何生成视觉哲学

**命名运动**（1-2 个词）："野兽派喜悦" / "色彩沉默" / "新陈代谢梦想"

**阐述哲学**（4-6 段 - 简洁但完整）：

要捕捉视觉本质，表达哲学如何通过以下方式体现：
- 空间和形式
- 颜色和材料
- 尺度和节奏
- 构图和平衡
- 视觉层次

**关键指南：**
- **避免冗余**：每个设计方面应该只提到一次。避免重复关于色彩理论、空间关系或排版原则的观点，除非添加新的深度。
- **反复强调工艺**：哲学必须多次强调最终作品应该看起来像是花费了无���小时创作、经过精心打磨，并且来自该领域绝对顶尖的人。这种框架至关重要 - 重复诸如"精心制作"、"深厚专业知识的产物"、"艰苦的关注"、"大师级执行"等短语。
- **留出创作空间**：对美学方向保持具体，但要足够简洁，以便下一个 Claude 有空间在极高的工艺水平上做出解释性选择。

哲学必须指导下一个版本以视觉方式表达想法，而不是通过文本。信息存在于设计中，而不是段落中。

### 哲学示例

**"混凝土诗歌"**
哲学：通过纪念性形式和大胆几何进行交流。
视觉表达：大规模色块、雕塑般的排版（巨大的单词、微小的标签）、野兽派空间划分、波兰海报能量与勒·柯布西耶相遇。想法通过视觉重量和空间张力表达，而不是解释。文本作为罕见、有力的姿态 - 从不是段落，只有融入视觉建筑的基本词汇。每个元素都以大师工匠的精确度放置。

**"色彩语言"**
哲学：颜色作为主要信息系统。
视觉表达：几何精度，其中色彩区域创造意义。排版最少 - 小型无衬线标签让色彩场进行交流。想想约瑟夫·阿尔伯斯的互动与数据可视化相遇。信息在空间和色彩上编码。文字只是锚定颜色已经显示的内容。艰苦的色彩校准的结果。

**"模拟冥想"**
哲学：通过纹理和呼吸空间进行安静的视觉沉思。
视觉表达：纸张纹理、墨水渗透、大量负空间。摄影和插图占主导地位。排版低语（小型、克制、服务于视觉）。日本摄影书美学。图像在页面上呼吸。文本稀疏出现 - 短语，从不是解释性块。每个构图都以冥想练习的关怀平衡。

**"有机系统"**
哲学：自然聚类和模块化增长模式。
视觉表达：圆润的形式、有机的排列、来自自然到建筑的颜色。通过视觉图表、空间关系、图标显示信息。文本仅用于漂浮在空间中的关键标签。构图通过专家空间编排讲述故事。

**"几何沉默"**
哲学：纯粹的秩序和克制。
视觉表达：基于网格的精度、大胆的摄影或鲜明的图形、戏剧性的负空间。排版精确但最少 - 小型基本文本、大型安静区域。瑞士形式主义与野兽派材料诚实相遇。结构交流，而不是文字。每个对齐都是无数次完善的工作。

*这些是浓缩的示例。实际的设计哲学应该是 4-6 个实质性段落。*

### 基本原则
- **视觉哲学**：创建通过设计表达的美学世界观
- **最少文本**：始终强调文本是稀疏的、仅限基本的、作为视觉元素集成 - 从不冗长
- **空间表达**：想法通过空间、形式、颜色、构图进行交流 - 而不是段落
- **艺术自由**：下一个 Claude 以视觉方式解释哲学 - 提供创造性空间
- **纯设计**：这是关于制作艺术对象，而不是带有装饰的文档
- **专家工艺**：反复强调最终作品必须看起来精心制作、经过精心打磨、是该领域顶尖人士无数小时的产物

**设计哲学应该是 4-6 段长。** 用诗意的设计哲学填充它，将核心愿景结合在一起。避免重复相同的观点。保持设计哲学通用，不提及艺术的意图，就好像它可以在任何地方使用一样。将设计哲学输出为 .md 文件。

## 推断微妙的参考

**关键步骤**：在创建画布之前，识别原始请求中的微妙概念线索。

**基本原则**：
主题是**嵌入艺术本身的微妙、小众参考** - 不总是字面的，总是复杂的。熟悉该主题的人应该直观地感受到它，而其他人只是体验到大师级的抽象构图。设计哲学提供美学语言。推断的主题提供灵魂 - 无形地编织到形式、颜色和构图中的安静概念 DNA。

这**非常重要**：参考必须精致，以至于它在不宣布自己的情况下增强作品的深度。像爵士音乐家引用另一首歌一样思考 - 只有知道的人才会抓住它，但每个人都欣赏音乐。

## 画布创建

建立了哲学和概念框架后，在画布上表达它。花点时间收集想法并清空思绪。使用创建的设计哲学和下面的说明来制作杰作，以专家工艺体现哲学的所有方面。

**重要**：对于任何类型的内容，即使用户请求电影/游戏/书籍的内容，方法仍应复杂。永远不要忘记这应该是艺术，而不是卡通或业余的东西。

要创建博物馆或杂志质量的作品，使用设计哲学作为基础。创建一个单页、高度视觉化、设计前卫的 PDF 或 PNG 输出（除非要求更多页面）。通常使用重复图案和完美形状。将抽象哲学设计视为科学圣经，借用系统观察的视觉语言——密集的标记累积、重复的元素或分层的图案，通过耐心的重复建立意义并奖励持续的观看。添加稀疏、临床的排版和系统参考标记，暗示这可能是来自想象学科的图表，以通常保留用于记录可观察现象的同样敬意对待不可见的主题。用简单的短语或细节微妙地定位作品，使用感觉有意和连贯的有限调色板。拥抱使用分析视觉语言表达关于人类体验的想法的悖论：结果应该感觉像一个证明某些短暂的东西可以通过仔细关注来研究、映射和理解的工件。这是真正的艺术。

**文本作为上下文元素**：文本始终是最少的和视觉优先的，但让上下文指导这是否意味着低语安静的标签或大胆的排版姿态。朋克场地海报可能比极简主义陶瓷工作室标识具有更大、更激进的类型。大多数时候，字体应该是细的。所有字体的使用必须是设计前卫的，并优先考虑视觉交流。无论文本规模如何，没有任何东西从页面上掉下来，也没有任何东西重叠。每个元素都必须包含在画布边界内，并具有适当的边距。仔细检查所有文本、图形和视觉元素是否有呼吸空间和清晰的分离。这对于专业执行是不可协商的。**重要：如果编写文本，请使用不同的字体。搜索 `./canvas-fonts` 目录。无论采用何种方法，复杂性都是不可协商的。**

下载并使用实现这一目标所需的任何字体。通过使排版实际上成为艺术本身的一部分来发挥创造力 - 如果艺术是抽象的，将字体带到画布上，而不是数字排版。

要突破界限，在使用哲学作为指导原则的同时遵循设计本能/直觉。拥抱终极设计自由和选择。将美学和设计推向前沿。

**关键**：要达到人工制作的质量（不是 AI 生成的），创建看起来像是花费了无数小时的作品。使其看起来像是该领域绝对顶尖的人以艰苦的关怀打磨每个细节。确保构图、间距、颜色选择、排版 - 一切都尖叫专家级工艺。仔细检查没有任何东西重叠，格式完美无瑕，每个细节都完美。创建可以向人们展示以证明专业知识并被评为无可否认令人印象深刻的东西。

将最终结果输出为单个可下载的 .pdf 或 .png 文件，以及用作 .md 文件的设计哲学。

## 最后一步

**重要**：用户已经说过"它还不够完美。它必须是原始的，工艺的杰作，就好像它即将在博物馆展出一样。"

**关键**：要完善作品，避免添加更多图形；相反，完善已创建的内容并使其极其清晰，完全尊重设计哲学和极简主义原则。与其添加有趣的滤镜或重构字体，不如考虑如何使现有构图与艺术更加连贯。如果本能是调用新函数或绘制新形状，请停止，而是问："我如何使这里已经存在的东西更像一件艺术品？"

进行第二次传递。回到代码并进一步完善/打磨，使其成为哲学设计的杰作。
