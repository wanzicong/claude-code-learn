# YouTube 视频笔记生成器

## 快速开始

用户提供 YouTube 视频链接后，按以下步骤生成笔记：

1. **提取视频信息**
   - 从 URL 中提取视频 ID
   - 使用 `scripts/get_transcript.py` 获取视频标题、字幕和元数据

2. **生成结构化笔记**
   - 分析字幕内容，识别核心主题和要点
   - 提取关键时间戳和对应内容
   - 识别重要概念和术语
   - 生成 Markdown 格式的完整笔记

3. **输出笔记**
   - 以中文输出笔记内容
   - 保存为 `.md` 文件（可选）

## 获取视频信息

提供三种获取方式，按推荐顺序使用：

### 方式一：youtube-transcript 库（推荐，最简单）

```bash
python scripts/get_transcript.py "<youtube-url>"
```

**安装依赖：**
```bash
pip install youtube-transcript
```

**特点：**
- 轻量级，安装简单
- 直接获取字幕数据
- 支持多语言和自动翻译

### 方式二：yt-dlp 库（备选，功能更强）

```bash
python scripts/get_transcript_ytdlp.py "<youtube-url>"
```

**安装依赖：**
```bash
pip install yt-dlp
```

**特点：**
- 功能更强大，支持更多网站
- 可获取章节信息
- 可下载视频元数据

### 方式三：无脚本模式（最后备选）

当无法安装 Python 库时，直接分析用户提供的信息：
- 请用户提供视频标题和描述
- 如果有字幕文件，请用户提供
- 基于可用的信息生成笔记

## 笔记结构

生成的笔记包含以下部分：

```markdown
# [视频标题]

> 来源：[YouTube 链接] | 频道：[频道名称] | 时长：[视频时长]

## 视频概述
[1-2 句话总结视频核心内容]

## 核心主题
- 主题 1
- 主题 2
- 主题 3

## 详细内容

### [时间戳] [章节/要点标题]
[该部分的详细内容摘要]

### [时间戳] [章节/要点标题]
[该部分的详细内容摘要]

## 重要概念与术语
| 术语 | 解释 |
|------|------|
| 概念1 | 解释内容 |
| 概念2 | 解释内容 |

## 关键要点
1. 要点一
2. 要点二
3. 要点三

## 总结与建议
[视频的总结和可行的建议]

## 原始字幕
[如有需要，可附上完整的原始字幕文本]
```

## 多语言处理

- 自动检测视频字幕语言
- 非中文字幕自动翻译为中文
- 保留原始语言的关键术语

## 处理特殊情况

### 无字幕视频
- 尝试使用自动生成字幕
- 如仍无字幕，告知用户并建议替代方案

### 长视频处理
- 超过 1 小时的视频，生成分段笔记
- 提供章节导航

### 教程类视频
- 额外提取步骤说明
- 生成操作清单

## 技术说明

### 支持的字幕获取库

**youtube-transcript**（方式一）
- 专门用于获取 YouTube 字幕
- 支持多语言字幕和自动翻译
- 轻量级，安装简单

**yt-dlp**（方式二）
- 强大的视频下载工具
- 支持更多视频网站
- 可获取章节信息和元数据

### 不同视频类型的笔记处理

根据视频类型选择合适的笔记结构：
- **教程类**：强调步骤和操作，参见 `references/note-patterns.md` 的"教程类视频笔记模式"
- **演讲类**：突出观点和故事线，参见"演讲/TED 视频笔记模式"
- **Vlog类**：突出体验和实用信息，参见"Vlog/生活方式视频笔记模式"
- **技术类**：突出代码和技术要点，参见"技术/编程视频笔记模式"
- **会议类**：突出议程和决议，参见"会议/讲座视频笔记模式"
