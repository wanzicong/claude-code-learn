# YouTube 视频笔记模式

本参考文档描述不同类型 YouTube 视频的笔记生成模式。

## 教程类视频笔记模式

教程类视频需要强调步骤和操作指南。

### 结构
```markdown
# [视频标题]

## 本教程涵盖内容
- 学习目标 1
- 学习目标 2

## 前置准备
- 工具/环境要求
- 基础知识要求

## 操作步骤

### 步骤 1：[步骤名称]
**时间戳：** [MM:SS]
**操作内容：**
- 具体操作 1
- 具体操作 2
**注意事项：** [重要提示]

### 步骤 2：[步骤名称]
...

## 关键代码/命令
```language
代码片段
```

## 常见问题
[视频中提到的常见问题和解决方案]

## 总结
- 完成后的效果
- 下一步学习建议
```

## 演讲/TED 视频笔记模式

演讲类视频需要突出观点和故事线。

### 结构
```markdown
# [演讲标题]
**演讲者：** [姓名] | **活动：** [会议/活动名称]

## 演讲核心观点
[一句话总结演讲的核心思想]

## 演讲大纲

### 开场引言 [时间戳]
- 演讲者背景
- 引入主题的方式

### 主要论点 1 [时间戳]
**观点：** [论点陈述]
**支撑故事/数据：** [具体内容]

### 主要论点 2 [时间戳]
**观点：** [论点陈述]
**支撑故事/数据：** [具体内容]

### 案例分享 [时间戳]
[具体案例描述]

### 结论与呼吁 [时间戳]
- 核心总结
- 行动呼吁

## 金句摘录
> "令人印象深刻的引用"

## 关键概念
| 概念 | 解释 |
|------|------|
| ... | ... |

## 思考与启示
[从演讲中获得的启发]
```

## Vlog/生活方式视频笔记模式

Vlog 类视频需要突出体验和实用信息。

### 结构
```markdown
# [视频标题]
**创作者：** [频道名称]

## 视频概览
[一句话描述视频内容]

## 时间线记录

### [地点/场景] [时间戳]
**活动内容：**
- 活动 1
- 活动 2
**亮点：** [特别值得关注的点]

### [地点/场景] [时间戳]
...

## 实用信息

### 提及的工具/产品
- [产品/服务名称] - 用途/说明

### 相关链接
[视频中提到的链接或资源]

## 个人感受/评价
[创作者表达的感受或观点]
```

## 调试与问题解决

| 错误 | 原因 | 解决方案 |

[建议进一步学习的方向]
```

```markdown

**主讲人：** [姓名] | **日期：** [日期]

## 议程概览
1. 议程项 1
2. 议程项 2

## 详细记录

### [议程项 1] [时间戳]
**主讲人：** [姓名]
**内容摘要：**
- 要点 1
- 要点 2
**讨论/问答：**
[重要的讨论内容或问答]

### [议程项 2] [时间戳]
...

## 关键决议/结论
1. 决议 1
2. 决议 2

## 行动项
| 行动项 | 负责人 | 截止日期 |
|--------|--------|----------|
| ... | ... | ... |

## 后续资源
[提到的相关文档、链接等]
```

## 通用笔记生成规则

无论视频类型如何，都应遵循以下规则：

1. **时间戳精度**：使用 [MM:SS] 或 [HH:MM:SS] 格式
2. **层级清晰**：使用 Markdown 标题层级组织内容
3. **重点突出**：使用粗体、列表、引用等格式突出重要信息
4. **简洁明了**：避免冗长描述，保留核心信息
5. **完整性**：涵盖视频的所有重要部分
6. **可操作性**：对于教程类视频，提供可执行的步骤清单
